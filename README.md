## Node.js

В проекте используется [Node.js](https://nodejs.org/) v14.19.0

## Скрипты для запуска
Запуск __dev__ режима и __build__
```
npm run dev
npm run build
```
Запуск сервера на localhost:3000
```
npm run start
```
